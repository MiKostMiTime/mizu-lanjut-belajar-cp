## ⚡ Quick Start

```bash
# Clone repo
git clone https://github.com/MiKostMiTime/mizu-lanjut-belajar-cp.git
cd mizu-lanjut-belajar-cp

# Mulai ngoding dengan template
cp templates/template.cpp solutions/cses/sorting-and-searching/namasoal.cpp
nvim solutions/cses/sorting-and-searching/namasoal.cpp
```
